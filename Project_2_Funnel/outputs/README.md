# E-commerce Conversion Funnel Analysis

## Objective
Analyze drop-offs across the e-commerce funnel and compare conversion performance by device.

## Tools
- Python
- Tableau Public

## What I Did
- Analyzed funnel stages from session start to order confirmation
- Calculated conversion rates across devices
- Identified high drop-off points in the checkout journey

## Key Insights
- Desktop users have the highest conversion rate
- Mobile users drop off heavily during checkout
- Improving mobile checkout UX can significantly lift conversions
